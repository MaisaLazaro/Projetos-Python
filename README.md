# Calculadora de IMC em Python
Script interativo desenvolvido em Python para cálculo do Índice de Massa Corporal (IMC), com classificação automática e opção de múltiplas execuções.

Este projeto tem como objetivo aplicar conceitos fundamentais de programação em Python, criando uma aplicação simples, funcional e interativa no terminal.

O usuário insere peso e altura, e o sistema retorna o valor do IMC juntamente com sua classificação.

## Funcionalidades:
* Entrada de dados via terminal;
* Cálculo automático do IMC;
* Classificação do resultado:
  * Abaixo do peso
  * Peso normal
  * Sobrepeso
  * Obesidade
* Execução contínua com opção de repetição.

## Lógica aplicada

O cálculo do IMC é realizado pela fórmula: IMC = peso / altura²
A classificação é definida com base em estruturas condicionais (`if/elif/else`), conforme faixas estabelecidas.

## Tecnologias utilizadas
* Python

## Principais aprendizados
* Estruturas de repetição (`while`);
* Condicionais (`if/elif/else`);
* Entrada e saída de dados;
* Organização de lógica de programação.
